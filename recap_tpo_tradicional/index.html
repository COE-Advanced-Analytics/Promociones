Salida: curvas demanda por SKU
Se construyen features de precio, volumen y contexto para cada SKU.
El modelo aprende la elasticidad precio–demanda usando datos históricos transaccionales.
Permite estimar el impacto esperado de aplicar diferentes niveles de descuento.
Se utiliza para simular promociones entre 2% y 8% antes de su ejecución.
Las curvas de demanda generadas optimizan la selección y timing de las tandas promocionales.
